# LabZero-Online-Lab-Visualization
LabZero is an interactive online platform for visualizing lab experiments and concepts in a virtual environment. It helps students learn by simulating real-world lab setups, making complex ideas easier to understand through hands-on, browser-based exploration.

Core Features:

User Authentication: Implemented a secure Login & Signup system to provide a personalized learning experience and track user progress.

Gesture-Based Interaction: Integrated a Gesture Controller allowing users to interact with 3D models using natural hand movements—perfect for an immersive, touchless laboratory experience.

Immersive 3D Library: Every module now features a dedicated 3D visualization window with manual rotation, zooming, and interactive data labels.

Offline Glossary: A local dictionary of scientific terminology for offline study.

The platform currently supports 4 core subjects, namely physics, chemistry, mathematics and biology with a growing library of interactive modules. 6 modules of chemistry, 2 modules each of physics, mathematics and biology are added along with their 3D interactive visualisation.
